# poc implements highchartsJS With React TS

Use:
```
Node: "12,14,01",
react: "16.13.1",
typescript: "3.7.5",
highcharts: "8.1.2",
highcharts-react-official: "3.0.0",
```

Run:

```
npm install
```
```
npm start
```
